# EODB - Haryana Land Records GIS Portal

Modern **React + Vite** frontend for Haryana government land records with **GIS mapping**.

## ✨ Features

- **Interactive GIS Map** (Leaflet + React Leaflet)
- **Haryana State/District/Tehsil/Village layers**
- **Responsive design** (Mobile + Desktop)
- **Modern UI** (Tailwind CSS + Lucide icons)
- **Ready for PostGIS/GeoServer integration**
